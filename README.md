## 🚀 Getting Started

### Prerequisites
- **Node.js** v18+
- **npm** or **yarn**

### Installation

```bash
# Clone the repository
git clone https://github.com/yashhhgoswami/yashhgoswami.git
cd yashhgoswami

# Install dependencies
npm install

# Start the development server
npm run dev
```

The site will be live at **`http://localhost:5173`**

### Build for Production

```bash
npm run build
npm run preview
```
